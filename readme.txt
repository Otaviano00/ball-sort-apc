Informações do projeto:

Execute o seguinte comando para rodar o projeto:

Windows: gcc game.c options.c utils.c data_base.c -o game  && .\game
Linux: gcc game.c options.c utils.c data_base.c -o game  && ./game

Extras implementados: 
- Modo Blind: 
    Mostra apenas as bolas do topo da coluna. Quando a coluna é completada, mostra a coluna toda.
    Por padrão, o jogo sempre inicia com o modo bliend desligado.
